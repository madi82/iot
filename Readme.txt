README

subject:-
This project is for those file which takes 10 to 20 min. to open.
If user wants to access or open desired file but it takes much time to open then,
user can remotly triggered computer from anywhere, and automaticaly open software,
so user dont have to wait for it. 

working-
Arduino helps to control computer via ethernet sheild, using website UI.
after turned on computer java code runs automatically and connect with server,
The java code retrive information so that computer knows which file to be open in desktop,
as per information retrived from server, java code triggered batch file, 
and using that batch code disired file will be open.
